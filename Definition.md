| **Antenna** | An antenna is a device that converts electrical signals into radio waves (for transmission) and vice versa (for reception). | 天线 | 一种将电信号转换为无线电波（用于发射）或将无线电波转换为电信号（用于接收）的装置。 | - |
| **Radiation Pattern** | An antenna radiation pattern or antenna pattern is defined as a graphical representation of the radiation properties of the antenna as a function of space coordinates. | 辐射图案 | 天线的辐射特性随空间坐标变化的图形表示。 | - |
| **Reactive Near-Field** | Area closest to the antenna, where the electromagnetic fields are highly reactive and non-radiating. | 反应近场区 | 最靠近天线的区域，电磁场以储能为主，不辐射能量。 | \( R < 0.62\sqrt{D^3/\lambda} \) |
| **Radiating Near-Field (Fresnel)** | The region between the reactive near-field region and the far-field region. The radiation starts to form in this region. | 辐射近场区（菲涅尔区） | 介于反应近场区与远场区之间的区域，辐射在此区域开始形成。 | \( R_1 < R < R_2 \) |
| **Far-Field (Fraunhofer)** | The area furthest from the antenna where electromagnetic fields are radiated. It's also known as the radiation zone or Fraunhofer region. | 远场区（夫琅禾费区） | 距离天线最远的区域，电磁场完全辐射出去，也称为辐射区。 | \( R > 2D^2/\lambda \) |
| **Isotropic Antenna** | Antenna which radiates equal amount of energy in all directions. | 各向同性天线 | 在所有方向上均匀辐射能量的理想天线模型。 | - |
| **Omnidirectional Antenna** | Radiates energy equally in all direction of horizontal plane but very limited radiation in vertical plane. | 全向天线 | 在水平面内均匀辐射，但在垂直面内辐射非常有限的天线。 | - |
| **Directivity** | Directivity (D) is a far field parameter which describes the ability of antenna to focus energy to a particular direction (or directions). | 方向性 | 描述天线将能量聚焦到特定方向的能力的远场参数。 | \( D = \frac{U_{\text{max}}}{U_{\text{avg}}} \) |
| **Gain** | Antenna gain is a measure of how well an antenna can focus energy to a particular direction compared to an isotropic antenna. | 增益 | 天线相对于各向同性天线在特定方向上集中能量的能力度量。 | \( G = e_{\text{tot}} \cdot D \) |
| **Impedance** | Impedance of an antenna refers to the relationship between the voltage and current at the feed point of the antenna. | 阻抗 | 天线馈电点处电压与电流之间的关系，通常表示为复数形式。 | \( Z = R + jX \) |
| **Reflection Coefficient (Γ)** | The reflection coefficient Γ is defined as the ratio of reflected voltage to incident voltage. | 反射系数 | 反射电压与入射电压之比，用于衡量阻抗失配程度。 | \( \Gamma = \frac{V_{\text{ref}}}{V_{\text{inc}}} = \frac{Z_L - Z_0}{Z_L + Z_0} \) |
| **Return Loss (RL)** | When Γ is expressed in dB it is known as return loss (RL). | 回波损耗 | 反射系数以分贝（dB）表示的形式，用于评估匹配性能。 | \( RL(\text{dB}) = -20\log_{10}|\Gamma| \) |
| **Half-wave Dipole Antenna** | One of the most basic and widely used types of antennas in radio and telecommunications. The length of a dipole antenna is half-wavelength. | 半波偶极天线 | 最基本且广泛使用的天线类型之一，其长度为工作波长的一半。 | \( L = \lambda/2 \) |
| **Microstrip Antenna** | A microstrip antenna is a type of antenna that consists of a radiating patch on one side of a dielectric substrate, with a ground plane on the other side. | 微带天线 | 一种由介质基板一侧的辐射贴片和另一侧的接地板构成的天线。 | - |
| **Friis Transmission Equation** | One of the fundamental equations in antenna theory used to calculate the power received from one antenna, when transmitted from another antenna, separated by a distance d, and operating at a frequency of f. | 弗里斯传输方程 | 天线理论中的基本方程，用于计算在自由空间中相距一定距离的两天线之间的接收功率。 | \( P_r = P_t G_t G_r \left( \frac{\lambda}{4\pi d} \right)^2 \) |
| **Parabolic Reflector Antenna** | Uses a curved surface with the cross-sectional shape of a parabola, to direct radio waves. | 抛物面反射天线 | 使用抛物线截面形状的曲面来导向无线电波的高增益天线。 | - |
| **Transponder** | A series of small, interconnected circuits that form a communications channel between the receiving and transmitting antennas of a satellite. | 转发器 | 卫星上连接接收和发射天线的一系列小型互连电路，构成通信信道。 | - |
| **5G** | 5G is the fifth generation of wireless cellular technology and is the new global wireless standard after 4G/LTE. | 5G | 第五代移动通信技术，是继4G/LTE之后的新一代全球无线标准。 | - |
| **eMBB** | Enhanced Mobile Broadband: Provides significantly faster data speeds and larger data volumes for mobile communications. | 增强移动宽带 | 提供显著更快的数据速率和更大数据容量的移动通信场景。 | - |
| **mMTC** | Massive Machine Type Communications: Supports the connectivity of a vast number of devices, particularly in massive IoT applications. | 大规模机器类型通信 | 支持海量设备连接，特别适用于大规模物联网应用。 | - |
| **URLLC** | Ultra-Reliable and Low Latency Communications: Designed for applications that require extremely low latency and high reliability. | 超可靠低时延通信 | 为需要极低时延和高可靠性的应用（如自动驾驶、远程手术）设计的通信场景。 | - |
| **AAS (Advanced Antenna System)** | An active antenna system, where the active transceiver array and the passive antenna array are intelligently integrated into a single hardware unit. | 先进天线系统 | 将有源收发阵列与无源天线阵列智能集成于一体的天线系统。 | - |
| **MIMO** | MIMO wireless setup is a combination of multiple transmit antennas at transmitter from which multiple inputs are provided to the wireless channel, and multiple receive antennas at the receiver in which multiple signals are received as the output of the wireless communication channel. | 多输入多输出 | 在发射端使用多个天线提供多个输入给无线信道，在接收端使用多个天线接收多个信号输出的无线通信技术。 | - |
| **Spatial Diversity** | Provides diversity gain. Multiple copies of the same data are sent across independently fading channels. | 空间分集 | 通过多个独立衰落的信道发送相同数据的副本，以获得分集增益，提高可靠性。 | - |
| **Spatial Multiplexing** | Provides multiplexing gain. Each channel carries independent information. | 空间复用 | 每个信道承载独立的信息，以获得复用增益，提高数据速率。 | - |
| **Massive MIMO** | Massive MIMO is a type of wireless communications technology in which base stations are equipped with a very large number of antenna elements, utilizing the same time and frequency resource, to improve spectral and energy efficiency. | 大规模MIMO | 一种基站配备极多天线单元，利用相同时频资源以提高频谱和能量效率的无线通信技术。 | - |
| **Beamforming** | Beamforming involves the directional transmission and reception of signals between a base station and user equipment. | 波束成形 | 基站与用户设备之间进行定向信号传输和接收的技术。 | - |
| **Multiple Access Technique** | Multiple access technique enables multiple users to simultaneously share a finite piece of radio spectrum. | 多址技术 | 使多个用户能够同时共享有限无线电频谱资源的技术。 | - |
| **BDMA (Beam Division Multiple Access)** | BDMA works by allocating highly directive orthogonal beams to multiple mobile stations in 5G networks. | 波分多址 | 通过为5G网络中的多个移动站分配高定向正交波束来实现多址接入的技术。 | - |
| **Beam Steering** | Beam steering is the process of dynamically changing the direction of the main lobe of a beam in an antenna array system. | 波束导向 | 通过调整天线阵列中各个单元的相位，动态改变波束主瓣方向的过程。 | - |
| **Phased Array Antenna** | A phased array antenna is a type of array antenna in which the phase of the signal at each individual antenna element is controlled electronically. | 相控阵天线 | 一种阵列天线，其每个天线单元的信号相位可通过电子方式控制，从而实现波束扫描。 | 阵列增益公式：\( G_{\text{array}} = 10\log_{10}(N) + G_e - L_{\text{ohmic}} - L_{\text{scan}} \) |
